richard drake


Degree in Mathematics, Pembroke College, Cambridge, 1979

Analyst/Programmer, Price Waterhouse, 1980-82

Co-founder, Objective Computer Systems, 1983

Objective was the first consultancy in Europe to specialise in object programming, the foundation of a number of modern programming languages, particularly Ruby, and influential on many others, including JavaScript, Python and PHP. As well as sales and recruitment, I was involved in project planning with users, based on Tom Gilb's pioneering agile methods, plus programming and testing for British Airports Authority, Bankers Trust, Reuters, TSB, Hill Samuel, Queensland Nickel Sales, Rio Tinto, Defence Research Agency, Chemical Bank and Sabre Fund Management. Languages: Java, Smalltalk, C++, Visual Basic, JavaScript and Python. Other tools: relational databases, from 1985, object databases, web systems, map-based user interfaces and xUnit unit testing.

Since 2005 I have been a consultant and developer for a variety of startups. Projects have included reevoo.com, timeindexing.com, partysuperstores.co.uk, wordia.com, memoori.com, osmosoft.com, ichauffeur.co.uk, livestation.com and b4bpayments.com. Languages: Ruby, JavaScript, CoffeeScript, Rust, PHP, Python and Java. Tools and services: Rails, Sinatra, Roda, MySQL, PostgreSQL, MongoDB, Redis, Node.js, Express, jQuery, QUnit, Mocha, RSpec, MiniTest, Capybara, WebDriver, Selenium, Pry, Byebug, ruby-prof, benchmark-ips, Pivotal Tracker, Trello, RVM, Rbenv, Jenkins, Travis, Heroku, AWS, PayPal gateway, KISSmetrics, Curebit, SendGrid, Airbrake, Exceptional, Subversion, Git, GitHub and SourceTree.
