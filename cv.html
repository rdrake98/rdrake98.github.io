richard drake


Degree in Mathematics, Pembroke College, Cambridge, 1979

Analyst/Programmer, Price Waterhouse, 1980-82

Co-founder, Objective Computer Systems, 1983

Objective was the first consultancy in Europe to specialise in object programming, the foundation of many modern programming languages, including Ruby and CoffeeScript, and influential on many others, including Python and PHP. As well as sales and recruitment, I was involved in project planning with users, based on agile methods, programming and testing for British Airports Authority, Bankers Trust, Reuters, TSB, Hill Samuel, Queensland Nickel Sales, Rio Tinto, Defence Research Agency, Chemical Bank and Sabre Fund Management. Languages: Java, Smalltalk, C++, Visual Basic, JavaScript and Python. Other tools: relational databases, from 1985, object databases, web systems, map-based user interfaces and xUnit unit testing.

Since 2005 I have been a consultant and developer for a variety of startups. Projects have included reevoo.com, timeindexing.com, wordia.com, livestation.com, memoori.com, osmosoft.com and ichauffeur.co.uk. Languages: Ruby, JavaScript, CoffeeScript, Rust, PHP, Python and Java. Tools and services: Rails, Sinatra, MySQL, PostgreSQL, MongoDB, Redis, Node.js, Express, jQuery, QUnit, Mocha, RSpec, MiniTest, Capybara, WebDriver, Selenium, Pry, byebug, ruby-prof, benchmark-ips, Pivotal Tracker, Trello, TextMate, RVM, Rbenv, Jenkins, Travis, Heroku, Amazon Web Services, PayPal gateway, KISSmetrics, Curebit, SendGrid, Airbrake, Exceptional, Subversion, Git, GitHub and SourceTree.
